# GAS App

## メール一括送信ツール

Google Apps Script（GAS）を使用したメール一括送信アプリケーションです。Googleスプレッドシートを使って、複数の宛先に一度にメールを送信できます。

### 機能

- 複数の宛先への一括メール送信
- 宛先ごとの個別挨拶文の自動挿入
- 添付ファイルの送信対応
- 送信前確認ダイアログ
- 送信結果の表示
- 送付先リストのクリア機能

### 注意事項

- Gmail API の制限により、1日あたりの送信数に制限があります
- 大量のメール送信を行う場合は、Googleの利用規約を確認してください
- 添付ファイルのサイズ制限にご注意ください（25MB以内）
